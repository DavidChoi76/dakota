Dakota console scripts
======================

These console scripts are called as Dakota's
`analysis_driver <https://dakota.sandia.gov//sites/default/files/docs/6.4/html-ref/interface-analysis_drivers.html>`_:
`dakota_run_component` for a CSDMS component,
and `dakota_run_plugin` for a model interfaced
by a Dakotathon `plugin class <model_plugins.html>`_.
